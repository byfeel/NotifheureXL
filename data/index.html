JSON MQTT
Topic Souscription

Json à envoyer sous la forme : {"msg":"Hello World"}

msg : message a afficher

Possible de préciser des options avec mot-clef opt : par exemple {"msg":"Hello World","opt":"pause=0; nzo=1"} pour une pause de 0 seconde sur la zone 1 ( séparé par ; ). Si aucune option n'est précisée .Les paramètres par défaut sont appliqués. Envoi du message sur la zone "message"

Topic Publication

Json sous la forme : 

    {
    "temperature": "20.70",     // Info température
    "humidity": "49.30",        // Info humidité
    "sec": true,                // Etat secondes
    "hor": true,                // Etat affichage horloge
    "lum": true,                // Mode luminosité
    "rev": false                // Etat réveil
    }
